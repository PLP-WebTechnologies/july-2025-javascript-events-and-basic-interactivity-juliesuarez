Frequently Asked Questions (FAQ)

What is DOM Manipulation?

The Document Object Model (DOM) is a programming interface for web documents. DOM Manipulation is changing the structure, style, and content of a web page using a language like JavaScript.

What is an Event Listener?

An event listener is a procedure or function in a computer program that waits for an event to occur. Examples of events are a user clicking a button, hovering over an element, or pressing a key.
